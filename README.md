# Neovim Configuration

A personal Neovim configuration built on top of [LazyVim](https://www.lazyvim.org/).

## About

This configuration extends LazyVim, a Neovim setup powered by [lazy.nvim](https://github.com/folke/lazy.nvim) that provides a good base with sensible defaults and easy plugin management.

## Installation

1. Make sure you have [Neovim](https://neovim.io/) v0.9.0 or higher installed
2. Back up your existing Neovim configuration (if any):
   ```bash
   mv ~/.config/nvim ~/.config/nvim.bak
   ```
3. Clone this repository:
   ```bash
   git clone https://github.com/cateperson/nvim.git ~/.config/nvim
   ```
4. Start Neovim:
   ```bash
   nvim
   ```
   LazyVim will automatically install all plugins on first launch.

## Usage

This configuration uses LazyVim's defaults. Press `<space>` to open the command menu and explore available keybindings.

For more information about LazyVim, visit the [official documentation](https://www.lazyvim.org/).
